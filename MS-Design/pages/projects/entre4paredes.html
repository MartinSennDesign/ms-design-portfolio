UX Design · Game Design

Entre 4 Paredes

UX design de videojuego mobile con mecánicas de sigilo. Proyecto universitario DG3.

Problema

Diseñar la experiencia completa de un videojuego mobile que funcione con mecánicas de sigilo, requiriendo sistemas de personajes coherentes, mapas jugables y una interfaz que no interrumpa la inmersión del jugador.

Solución

Pixel art original de personajes y enemigos, sistema de diseño de UI para el juego, mapas de niveles detallados y flowchart completo de jugabilidad. Cada elemento fue diseñado con coherencia visual y funcional.
